# KeypressOSD
Display key press on screen

**(since v2.62)**

Optionally save pressed keys to log file in: `C:\Users\{USER}\AppData\Roaming\KeypressOSD\`
Allow quick enabling by typing the following:
- `###1` Enable GUI
- `###2` Disable GUI
- `###3` Enable Logging
- `###4` Disable Logging
